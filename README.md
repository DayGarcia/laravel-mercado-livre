# Mercado Livre Laravel

<h4>A package to connect to Mercado Livre API</h4>

## Requirements

PHP 8 and later

## Installation & Usage

### Composer

To install via [composer](http://getcomposer.org/):

```
composer require daygarcia/laravel-mercado-livre
```
